# Ferdinando Balzarro — L'Uomo prima del Maestro

Personal website for Ferdinando Balzarro: Maestro VIII Dan, writer, former competitive parachutist.
Built with Astro, deployed on GitHub Pages.

## Local development (optional)
```bash
npm install
npm run dev
```

## Content management
- Books: `src/data/books.json`
- Schedule: `src/data/schedule.json`
- Quotes: `src/data/quotes.json`
- Images: `public/images/`

## Deploy
Push to `main` → GitHub Actions builds & deploys to `https://ferdinandobalzarro.it`
